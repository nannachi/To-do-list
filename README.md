# 📝 To-Do List App

Una semplice applicazione To-Do sviluppata con **HTML, CSS e JavaScript**, creata per mettere in pratica la manipolazione del DOM e la gestione degli eventi.

##  Funzionalità

-  Aggiunta di nuove attività
-  Inserimento rapido tramite tasto *Invio*
-  Eliminazione delle attività
-  Contrassegnare le attività come completate (checkbox)
-  Contatore dinamico (attività completate / totali)
-  Validazione input con messaggio di errore
-  Animazione fluida durante l’eliminazione delle task

## Tecnologie utilizzate

- HTML5  
- CSS3  
- JavaScript
